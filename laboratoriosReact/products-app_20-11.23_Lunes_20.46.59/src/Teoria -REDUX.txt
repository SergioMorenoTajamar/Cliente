/**
*
*  TERMINOS IMPORTANTES EN REDUX
*
*/


 -  Action: 
			Operación que permite la carga de datos en el almacén.
			Redux no permite el manejo de datos directametne y precia de [actions]
			para especificar qué cambios queremos hacer.

 -  Action Type: 
			Objetos de Javascript con un parámetro que especifíca el tipo de action

 -  Action Creator: 
			Función que crea un [action]. Los componentes la reconocen porque es 
			{funtion prop}.
 -  Reducer: 
			Función que recibe el [action] y gestiona los cambios en el data store.
			El [action] especifica la operación a realizar pero es el reducer el que
			se encarga de ella
-   selector:
			Proporciona el componente con accesoal data store. 
			Los componentes los reconoce como {data props}
